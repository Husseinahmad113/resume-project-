Project 3: Weather App with API Integration

For this project, I created a Weather Application that fetches real-time weather data using a public API. The app allows users to search for a city and view current conditions like temperature, humidity, and weather description.

What it Does

  • Accepts city input from the user
  • Connects to a weather API to live weather data
  • Displays temperature, humidity, and conditions in a clean layout

Technologies that would be Used

  • Python
  • HTML, CSS, JavaScript (for frontend)
  • OpenWeatherMap API
